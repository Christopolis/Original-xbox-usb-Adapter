I designed this pcb to fit snug inside the enclosure that "emaxx620" Published [here: ](https://makerworld.com/en/models/513110-usb2ogxbox-enclosure#profileId-429212)
my printer was printing a little small, so i printed the enclosure at 101.2% scale.

I used [this](https://www.digikey.ca/en/products/detail/gct/USB1130-15-A/13545899) USB port, but i trimmed all the plastic support pieces off that were under it in order for it to sit way more flush with the board.
you may be able to use other usb ports, but this is the one i used. Please let me know if there's a usb you can buy that doesn't require trimming and sits flush to the pcb


JLC ORDER DETAILS:

the only important thing here is you don't make your pcb too thick so that it'll fit in the enclosure. i used 1.0mm pcb's, but had to add solder to the contacts, because they weren't making contact with the connectors without it.
again please contact me if you try a larger pcb size and it works well for you.


sometimes when inserting the pcb into the enclosre it can be a tight fit, but that's by design.
